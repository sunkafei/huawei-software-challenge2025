源代码放在了huawei-software-challenge2025文件夹下面：

| 文件名            | 说明           |
| ----------------- | -------------- |
| main.cpp          | 核心算法       |
| utility.h         | 通用工具函数   |
| hyperparameters.h | 算法超参数     |
| constants.h       | 定义了一些常量 |
| generator.py      | 数据生成器     |
| debug.py          | 错误检测脚本   |
| finetune.py       | 超参数搜索脚本 |

算法解析请参见[知乎@孙咖啡](https://zhuanlan.zhihu.com/p/1898417302979981487).
